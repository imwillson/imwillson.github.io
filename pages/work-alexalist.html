Built a MEAN stack application with an Echo Dot voice interface to add items to an online marketplace that we built

Used: Javascript. MongoDb, Express, Angular, Node.js, AWS Lambda, Amazon Echo Dot

Context

Our team wanted to work with hardware but not something so low-level that it would take too much time to learn. So we created the simple idea based around the Amazon Echo. We also wanted to practice building with MEAN again.

Worked with partners Nicholas Jativa and Jocelyn Ao

Components

Basic homescreen where one user could add and delete through Amazon echo

Amazon Echo Dot AWS, merged with Amazon Skills Portal, to help with processing our voice requests to add an item or delete from our forum

What I learned:

Using Amazon Echo Dot Skills Portal was clunky, but we soon figured out that in order for us to run a certain function, we had to use certain trigger words. Which limited our application to a few specific words.

A big challenge we ran into was working with AWS for the time time. It took us awhile to figure out how to utilize the JS modules within AWS. This was important to our project because we wanted to modify our transactions with Alexa's processsing.
